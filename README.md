# Fast ava node exploding

### Prerequisites

`sudo apt install git`

Tested on Ubuntu 18.04 but other linux distros should be OK also.

Go to home directory and use commands below.

```
git clone https://github.com/k0kk0k/ava-bundle.git
cd ava-bundle
chmod +x config.sh
./config.sh
cd ~
./avash
```

enjoy!
